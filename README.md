# cmeth
Computational Methods project

## Installation
First, call
```bash
$ make install
```
to create all the directories needed for the building;
then, call
```bash
$ make <test_name>
```
with the name of the unit test to be compiled.


To **uninstall**, simply type
```bash
$ make uninstall
```
